=================

Format: 1080p MP4 (H.264 + AAC), looped-friendly, 15-30s, no audio reliance
       (videos auto-play muted). Filesize ideally < 8 MB each.

If a brand video file is missing, the hero image (poster) shows in its place
with no error — no other code changes needed.
